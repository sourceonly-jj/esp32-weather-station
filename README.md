# ESP32 WiFi Weather Station

A small IoT weather station built using an ESP32.

Features
- Temperature & humidity monitoring
- NTP time synchronisation
- OLED display output

Hardware
- ESP32
- DHT11
- OLED display

Future improvements
- e-ink display
- data logging
- enclosure

Libraries to download
-Adafruit GFX
-Adafruit SSD1306
-DHT sensor library
